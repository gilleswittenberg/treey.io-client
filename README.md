## Development
- `git clone https://github.com/gilleswittenberg/treey.io-client.git`
- `npm install`
- `npm test`
- `npm run eslint`
- `npm run flow`
- `npm run dev-server`
- `npm run dev`
- open [localhost:8080](http://localhost:8080) in your browser
