# keyboard shortcuts

- ↓, tab            activate visual next node
- ↑, shift + tab    activate visual previous node
- →                 expand active node
- ←                 collapse active node
- enter             edit / add
- esc               exit editing / adding
- D                 delete
- +                 add child
- cmd + Z           undo

# keyboard + mouse

- cmd + click       select / active
- alt + click       edit
- select drag       select / active multiple
- shift + ↓, ↑      select / active multiple
